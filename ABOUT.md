# О проекте

## Pocket-TTS-Spokenword (Русская версия)

Генератор аудиокниг с эмоциональным анализом для русского языка.

### Ключевые особенности
- 🇷🇺 Полная поддержка русского языка
- 🎭 Автоматический анализ эмоций (RuBERT)
- 🎤 5 русских голосов (Silero TTS)
- 💻 Работает на CPU (не требует GPU)
- 🖥️ Графический интерфейс на русском
- 📦 Портативная установка для Windows

### Технологии
- **TTS**: Silero TTS (русский)
- **Эмоции**: RuBERT (cointegrated/rubert-tiny2-cedr-emotion-detection)
- **GUI**: PyQt5
- **Python**: 3.10-3.12

### Форк
Этот проект является форком [Pocket-TTS-Spokenword](https://github.com/danneauxs/Pocket-TTS-Spokenword), адаптированным для русскоязычной аудитории.
