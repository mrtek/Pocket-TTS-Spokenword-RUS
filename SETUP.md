# Инструкция по переустановке для поддержки русского языка

## Что было изменено

Программа была адаптирована для работы с русским языком:

1. **TTS модель**: Заменена Pocket TTS на Silero TTS для русского языка
2. **Emotion analyzer**: Заменена английская модель на RuBERT для русских эмоций
3. **Text processing**: Добавлена поддержка русских паттернов глав
4. **Конфигурация**: Обновлены маппинги эмоций и ключевые слова

## Шаги переустановки

### 1. Удалите старое окружение

```batch
rmdir /s /q .venv
rmdir /s /q .portable
rmdir /s /q ASR\venv
```

### 2. Запустите установку заново

```batch
install.bat
```

Это установит:
- Python 3.12 (портативно)
- Silero TTS для русского языка
- RuBERT для анализа эмоций
- Все обновленные зависимости

### 3. Запустите программу

```batch
start.bat
```

## Изменения в использовании

### Поддерживаемые языки
- ✅ **Русский язык** (основной)
- ❌ Английский язык (больше не поддерживается)

### Доступные голоса (Silero TTS)
- `baya` (женский, по умолчанию)
- `aidar` (мужской)
- `kseniya` (женский)
- `xenia` (женский)
- `eugene` (мужской)
- `random` (случайный)

### Поддерживаемые эмоции
- Радость (joy)
- Грусть (sadness)
- Удивление (surprise)
- Страх (fear)
- Гнев (anger)
- Нейтральность (no emotion)

### Паттерны глав (русский)
Программа теперь распознает:
- Глава 1, ГЛАВА 1
- Часть 1, ЧАСТЬ 1
- Раздел 1, РАЗДЕЛ 1
- Книга 1
- Пролог, Эпилог, Введение

## Важные изменения

### Voice Cloning
⚠️ **Silero TTS не поддерживает voice cloning**

Вместо загрузки своего голоса, выбирайте один из предустановленных голосов.

### Скорость генерации
- Silero работает быстрее чем Pocket TTS
- Оптимизирован для CPU
- Sample rate: 48000 Hz (вместо 24000 Hz)

### Качество аудио
- Высокое качество русской речи
- Естественная интонация
- Поддержка эмоциональной окраски через скорость речи

## Проверка установки

После установки проверьте:

1. Запустите `start.bat`
2. Загрузите русский текстовый файл
3. Выберите голос (например, `baya`)
4. Нажмите "Generate"

При первом запуске Silero TTS скачает модель (~100MB) из torch hub.

## Устранение проблем

### Ошибка при загрузке модели
```
Failed to load Silero TTS model
```
**Решение**: Проверьте интернет-соединение, модель скачивается с GitHub

### Ошибка emotion analyzer
```
Failed to load emotion model
```
**Решение**: Модель RuBERT скачивается с Hugging Face, проверьте доступ

### Программа не запускается
**Решение**: 
1. Удалите `.venv` и `.portable`
2. Запустите `install.bat` заново
3. Убедитесь что используется Python 3.12 (не 3.13+)

Если нужно вернуться к английской версии:

2. Переустановите:
```batch
rmdir /s /q .venv

```
